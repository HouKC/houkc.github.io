## 注意

注意开服务器不要用太小的端口号，也不要在家目录直接开，找个安全点的目录打开命令行或者终端开启。



## php版

php5.6以上环境下，命令行执行

```
php -S 0.0.0.0:9000
```



## python2版

```
python -m SimpleHTTPServer 9000
```



## python3版

```
python3 -m http.server 9000
```
